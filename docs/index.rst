Welcome to rapidsms-celery-router!
==================================

rapidsms-celery-router is a RapidSMS_ router implementation that uses Celery_
to queue incoming and outgoing messages.

.. _RapidSMS: http://www.rapidsms.org/
.. _Celery: http://www.celeryproject.org/
